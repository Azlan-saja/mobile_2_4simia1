Buka project terbaru
================
1.  Akses "github.com/Azlan-saja"
	=> Klik menu "Repositories"
	=> Cari dan klik "mobile_2_4simia1"
	=> Klik "CODE" dan salin HTTPS url
2. Buka menggunakan "flutlab.io"
	=> Akses "FlutLab.io" dan klik "Sign In" => "Continue With Google"
	=> Klik "New Project" selanjutnya klik "Import From VSC"
	=> Klik "Public Repo"
	=> Repo URL => Tempel hasil salin HTTPS url diatas
	=> Project Name => "Pertemuan4" selanjutnya klik import.
3. ATAU buka menggunakan VS CODE
	=> Klik menu "VIEW" dan pilih "Command Palette"
	=> Ketikkan "Git" dan pilih "Git: Clone"
	=> Tempel hasil salin HTTPS url diatas dan ENTER
	=> Tentukan lokasi simpan (Contoh di local disk D)
	=> Cari file "pubspek.yml", klik kanan pilih "Get Package"
